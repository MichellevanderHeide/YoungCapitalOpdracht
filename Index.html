Tabel met HTML tags en CSS keys

HTML tag/css Functie Uitvoering
p Met de tag p kun je een paragraaf toevoegen In de tekst "Lorem Ipsum is gebruik gemaakt van de tag p
h1 Met de tag h1 kun je titels maken h1 is de belangrijkste en h6 de minst belangrijke h1 is gebruikt voor Kennisopdracht Michelle van der Heide
h2 Met de tag h2 kun je titels maken, deze titel is iets minder belangrijk dan h1 h2 is gebruikt voor de tetel van deze tabel
br Met de tag br kun je zinnen opbreken br is gebruikt bij de tekst Lorem Ipsum
hr Met trek je een horizontale break lijn hr is gebruikt onder het menu
div Met de div tag sluit je een deel "division" van een HTML document in div is gebruikt voor de foto van de brug in Nieuw-Zeeland en voor het drop-up menu
table Met de table tag kun je een tabel toevoegen Table is gebruikt om deze tabel te maken
tr Een tr tag gebruik je als rij in een tabel Tr is gebruikt om deze tabel te maken
td De td tag gebruik je om informatie in een tabel te verwerken Deze zin staat tussen td tags
ul De ul tag wordt gebruikt om een lijst (ongeordend) met opsommingstekens te maken De ul tag is gebruikt in de opsomming onderaan deze pagina
li De Li tag wordt gebruikt om elementen in een lijst weer te geven De li tag is gebruikt in de opsomming onderaan deze pagina
a De a tag wordt gebruikt om een hyperlink toe te voegen aan je pagina De a link wordt gebruikt om je door te verwijzen naar de website van Fleetwood Mac onderaan deze pagina
input De input tag wordt gebruikt voor wanneer de gebruiker informatie moet geven De input tag wordt gebruikt om informatie op te vragen bij de gebruiker en staat onderaan deze pagina
button De button tag wordt gebruikt om knoppen te maken De button tag is gebruikt om een alert weer te geven (zie de knop onderaan de pagina)
image De image tag wordt gebruikt om plaatjes toe te voegen aan je pagina De image tag is gebruikt om de header en de brug in Nieuw-Zeeland toe te voegen
select De select tag wordt gebruikt om een lijst weer te geven wanneer je op een knop klikt De select tag wordt gebruikt om een lijst weer te geven, deze is onderaan deze pagina te vinden
option De option tag wordt gebruikt om een optie weer te geven in de eerder genoemde select lijst De option tag wordt onderaan deze pagina gebruikt
background-color De CSS key background-color wordt gebruikt om een kleur toe te voegen aan de achtergrond De CSS key background-color is hier gebruikt om de achtergond een blauwe kleur te geven
color De CSS key color wordt gebruikt om iets een kleur te geven De CSS key color is bijvoorbeeld gebruikt om de titel "De website" een groene kleur te geven
font-size De CSS key font-size wordt gebruikt om het formaat aan te passen van een lettertype De CSS key font-size is bijvoorbeeld gebruikt om bij het menu aan te geven hoe groot het lettertype is
padding De CSS key padding creërt ruimte voor bepaalde content aangeduid in pixels (boven,onder,link en rechts) De CSS key padding is gebruikt bij het doorzichtige blok boven de header
margin De CSS key margin creërt ruimte om een object heen aangeduid in pixels De CSS key margin is gebruikt in de foto "brug in Nieuw-Zeeland"
font De CSS key font kun je gebruiken om bepaalde lettertypes te kiezen voor een stuk tekst De CSS key font is gebruikt voor de titel "Kennisopdracht Michelle van der heide"
border Met de CSS key border kun je objecten/teksten omlijnen Bij de titel "HTML tags en CSS keys" is gebruik gemaakt van een "dashed" omlijning
